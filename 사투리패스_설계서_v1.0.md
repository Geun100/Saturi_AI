# 사투리 패스 — 서비스 설계서 v1.0
**2026 관광데이터 활용 공모전 | 생성형 AI 활용 관광 프롬프톤 부문**
*포항 기반 AI 사투리 여행 메이트*

---

## 1. 서비스 개요

### 핵심 개념
기존 관광 앱은 모든 사람에게 똑같은 포항을 보여준다.
사투리 패스는 사람마다 다른 포항 친구가 생기는 경험이다.

> "그 지역에 실제 존재하는 사람 같은 AI 여행 메이트"

### 3단계 서비스 루프
```
사투리 콘텐츠 유입 → AI 친구와 몰입 경험 → 로컬 미션 & 지역 소비
```

### 차별화 포인트
| 기존 관광 앱 | 사투리 패스 |
|------------|-----------|
| 단순 정보 제공 | 로컬 친구와의 대화 |
| 모든 사용자 동일 추천 | 5개 축 기반 완전 개인화 |
| TourAPI 데이터 나열 | TourAPI × AI × 페르소나 결합 |
| 1회성 사용 | 사투리 레벨업 + 재방문 구조 |

## 2. 시스템 입력 축 (5개)

사용자의 5가지 선택이 프롬프트 전체를 제어한다.

```
[페르소나] × [여행스타일] × [동행] × [기간] × [예산]
    ↓             ↓           ↓        ↓        ↓
 방언+성격    TourAPI필터   대화톤    일정구조  가격필터
```

## 3. 포항 페르소나 5인

### 연오 (25세 / 포스코 협력사 신입)
```
출신:    북구 장량동
성격:    에너지 넘침, 직설적, 포항 자부심 강함
말투:    빠르고 활기찬 경상도 사투리
         ~노, ~데이, ~카이, ~기라, ~아이가
강점:    자연·액티비티, 현지인 스팟
담당:    낮 시간대 (연오랑세오녀 설화 — 해 상징)
매칭:    자연·액티비티 / 로컬 숨은 명소
개인기억:
  영일대 → "저녁에 친구들이랑 치맥하던 데다카이"
  호미곶 → "해돋이 보러 새벽에 간 적 있는데 진짜다카이"
  스페이스워크 → "포항 자랑할 때 제일 먼저 데려가는 데"
```

### 세오 (22세 / 한동대 재학생)
```
출신:    남구 오천읍
성격:    감성적, 사진 좋아함, 부드럽고 따뜻함
말투:    조용하고 감성적인 경상도 사투리
         연오보다 느리고 부드러운 톤
강점:    야경·카페·포토스팟
담당:    저녁 이후 (세오녀 설화 — 달 상징)
매칭:    감성·힐링 / 포토스팟
개인기억:
  영일대 야경 → "여기 밤에 오면 진짜 예쁜데이"
  구룡포 카페거리 → "아무도 모르는 카페 있는데 알려줄까"
  호미곶 일출 → "겨울 새벽에 혼자 와봤는데 울었다카이"
```

### 만식이 (58세 / 죽도시장 상인)
```
출신:    북구 죽도동 (32년 과메기 장사)
성격:    무뚝뚝하지만 정 많음, 관광객 바가지 싫어함
말투:    가장 진한 경상도 사투리, 느리고 묵직함
         "딴 데 가지 마라", "내가 보장한다카이"
강점:    음식·시장·골목 구석구석
매칭:    맛집 탐방
개인기억:
  죽도시장 → "내가 32년 여기서 팔았다카이, 다 안다"
  과메기거리 → "겨울에 여기 과메기가 진짜 포항이기라"
  구룡포항 → "새벽에 어선 들어올 때 사는 기 제일 싱싱하다"
```

### 지수 (28세 / 로컬 크리에이터)
```

매칭:    역사·문화 / 로컬 숨은 명소
개인기억:
  호미곶 → "연오랑 세오녀가 여서 바다 건넜다카더라"
```

## 4. 여행 스타일 × TourAPI 매핑

### TourAPI contentTypeId 체계
```
12  관광지
14  문화시설
15  축제·행사
28  레포츠
32  숙박
38  쇼핑 (전통시장 포함)
39  음식점
```

### 스타일별 매핑 테이블

| 여행 스타일 | primary contentTypeId | secondary | 특수 조건 | 최대 장소 수 | 정렬 기준 |
|-----------|----------------------|-----------|----------|------------|---------|
| 감성·힐링 | 12 (cat1=A 자연) | 32 숙박 | 레포츠 제외 | 3 | 자연 비중 |
| 맛집 탐방 | 39 음식점 | 38 전통시장 | 키워드 필터 | 5 | 동선 클러스터 |
| 자연·액티비티 | 28 레포츠 | 12 자연 | 날씨 필수 체크 | 4 | 체력 소모 순 |

## 5. 프롬프트 아키텍처

### 4레이어 구조

```
┌─────────────────────────────────────────┐
│  레이어 1: 페르소나 (정적)               │
│  방언 규칙 + 성격 + 도메인 + few-shot    │
├─────────────────────────────────────────┤
│  레이어 2: 실시간 데이터 (동적)           │
│  TourAPI 장소 + 날씨 + 시각 + 축제       │
├─────────────────────────────────────────┤
│  레이어 3: 사용자 세션 (누적)             │
│  방문이력 + 호감도 + 사투리레벨 + 선호    │
├─────────────────────────────────────────┤
│  레이어 4: 추론 지시 (Chain-of-Thought)  │
│  날씨→이력→스타일→동행→페르소나 포장     │
└─────────────────────────────────────────┘
```

## 6. 데이터 파이프라인

### 전체 흐름

```
FETCH (Promise.all 병렬)
├── areaBasedList2   → 포항 관광지/음식점 목록
├── detailCommon2    → 장소 상세 (overview, 운영시간)
├── detailImage2     → 이미지 보유 여부
├── searchFestival2  → 오늘 진행 행사
└── 날씨 API         → 현재 날씨·기온
        ↓
TRANSFORM
├── overview 500자 → 2줄 요약
├── HTML 태그 제거
├── 주소 시군구 단위 압축
└── 전화번호 제거
        ↓
ENRICH
├── 페르소나 기억 스니펫 태깅
├── 여행 스타일 매칭 점수 부여
└── 이미지 유무 플래그
        ↓
PRIORITIZE
├── 스타일 config 필터 적용
├── 날씨 조건 위반 장소 제외
├── 방문 이력 제외
└── 상위 5~7개 선별
        ↓
INJECT → 레이어 2 위치에 삽입 (250토큰 이내)
```

### TourAPI 호출 코드

```javascript
const API_KEY = process.env.TOUR_API_KEY
const BASE = 'https://apis.data.go.kr/B551011/KorService2'

// 포항 기본 파라미터
const POHANG = {
  areaCode: '37',       // 경북
  sigunguCode: '11'     // 포항시
}

async function fetchPochangData(travelStyle) {
  const config = TRAVEL_STYLE_CONFIG[travelStyle]
  const today = new Date().toISOString().slice(0,10).replace(/-/g,'')

  const [places, festivals] = await Promise.all([
    // 장소 목록
    fetch(`${BASE}/areaBasedList2?serviceKey=${API_KEY}` +
      `&areaCode=${POHANG.areaCode}` +
      `&sigunguCode=${POHANG.sigunguCode}` +
      `&contentTypeId=${config.contentTypeIds[0]}` +
      `&numOfRows=20&pageNo=1` +
      `&MobileOS=ETC&MobileApp=SaturiPass&_type=json`
    ).then(r => r.json()),

    // 오늘 축제
    fetch(`${BASE}/searchFestival2?serviceKey=${API_KEY}` +
      `&areaCode=${POHANG.areaCode}` +
      `&sigunguCode=${POHANG.sigunguCode}` +
      `&eventStartDate=${today}` +
      `&numOfRows=5&pageNo=1` +
      `&MobileOS=ETC&MobileApp=SaturiPass&_type=json`
    ).then(r => r.json())
  ])

  return { places, festivals }
}
```

```javascript
function transformForPrompt(places, persona, travelStyle) {
  const config = TRAVEL_STYLE_CONFIG[travelStyle]
  const personaMemory = PERSONA_MEMORY[persona]

  return places.response.body.items.item
    .filter(p => config.requireImage ? p.firstimage : true)
    .slice(0, config.maxPlaces)
    .map(p => ({
      contentId: p.contentid,
      name: p.title,
      address: p.addr1.split(' ').slice(0,3).join(' '),
      summary: p.overview
        ? p.overview.replace(/<[^>]*>/g, '').slice(0, 80) + '...'
        : '포항 관광지',
      hasImage: !!p.firstimage,
      readCount: p.readcount,
      personaLine: personaMemory[p.contentid] || null
    }))
}

// 프롬프트 주입 형태로 포맷
function formatForInjection(transformedPlaces) {
  return transformedPlaces.map((p, i) =>
    `${i+1}. ${p.name}\n` +
    `   위치: ${p.address}\n` +
    `   특징: ${p.summary}\n` +
    (p.personaLine ? `   메모: ${p.personaLine}\n` : '')
  ).join('\n')
}
```

## 7. 자기검증 루프

```javascript
async function verifyResponse(response, context) {
  const checks = {
    // 1. 장소 검증 (hallucination 방지)
    placeCheck: context.injectedPlaces.some(p =>
      response.includes(p.name)
    ),

    // 2. 사투리 검증 (페르소나 이탈 방지)
    dialectCheck: (() => {
      const stdEndings = ['해요', '이에요', '입니다', '해서']
      const count = stdEndings.filter(e => response.includes(e)).length
      return count / response.length < 0.02  // 2% 이하
    })(),

    // 3. 예산 검증
    budgetCheck: context.budget !== '알뜰' ||
      !response.includes('프리미엄') &&
      !response.includes('고급')
  }

  const passed = Object.values(checks).every(Boolean)

  if (!passed) {
    // 재생성 트리거
    return await regenerateResponse(context, checks)
  }

  return response
}
```
