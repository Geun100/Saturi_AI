# 사투리 패스 (Saturi Pass)
### 2026 관광데이터 활용 공모전 출품작

> 포항에 사는 AI 친구와 함께하는 경상도 사투리 여행 메이트

---

## 서비스 소개

**사투리 패스**는 포항 현지인 AI 페르소나 5인이 경상도 사투리로 여행자를 안내하는 모바일 웹 앱입니다.
단순 관광 정보 제공을 넘어, 사용자의 여행 스타일에 맞는 맞춤 코스를 대화형으로 추천합니다.

---

## 핵심 기능

| 기능 | 설명 |
|---|---|
| 💬 AI 사투리 채팅 | 5개 페르소나가 경상도 사투리로 여행 추천 |
| 🗺️ 실시간 지도 연동 | 대화 중 언급된 장소를 지도에 자동 하이라이트 |
| 🏆 미션 & 포인트 | 장소 방문·퀴즈·챌린지 완료 시 포인트 적립 |
| 🎫 상품권 교환 | 적립 포인트로 포항 지역 상품권 발급 |
| 📍 관광 데이터 API | 한국관광공사 Tour API 기반 포항 명소 데이터 |

## 기술 스택

- **Frontend**: Next.js 16, React 18, TypeScript, Tailwind CSS
- **AI**: Anthropic Claude API (claude-sonnet-4-6)
- **지도**: Leaflet.js + CartoDB 다크 타일
- **데이터**: 한국관광공사 Tour API (포항 관광지 데이터)

---

## 로컬 실행

```bash
# 패키지 설치
npm install

# 환경변수 설정 (.env.local)
ANTHROPIC_API_KEY=your_key
TOUR_API_KEY=your_key

# 개발 서버 실행
npm run dev
# → http://localhost:3000
```
